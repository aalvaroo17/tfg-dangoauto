# DangoAuto - Aplicación Android

Aplicación móvil para el concesionario DangoAuto desarrollada en Android Studio con Java.

## Estructura

- **MainActivity**: Pantalla principal con 3 botones principales
- **SearchActivity**: Búsqueda y filtrado de coches con RecyclerView
- **SellActivity**: Formulario para vender coches

## Requisitos

- Android Studio Hedgehog o superior
- Android SDK 24+ (Android 7.0)
- Java 8+

## Instalación

1. Abre el proyecto en Android Studio
2. Sincroniza Gradle
3. Ejecuta la aplicación en un emulador o dispositivo físico
